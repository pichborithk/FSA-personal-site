Gradient Color Generator

This project is built using HTML, CSS, and JavaScript.
Here are some key features of "Gradient Color Generator":

  • Randomly generates background gradient colors
  • Displays the color code for each generated gradient
  • Easy to use - simply hit the generate button to create a new gradient color
  • Flexible - can be used for a wide range of design projects, from website backgrounds to graphic design work
With "Gradient Color Generator" you can easily find the perfect color scheme for your project, without the hassle of manually selecting and mixing colors. Whether you're a designer, developer, or just looking to add some color to your website, "Gradient Color Generator" has got you covered. It's an essential tool for any designer's toolbox.

Random Quote

This project is built using HTML, CSS, JavaScript and API.
Here are some key features of "Random Quote":

  • Generates random quotes from a database of quotes via API call
  • Changes the background color of the page with each quote
  • Option to share the quote on social media
  • Responsive design, works on any device
With "Random Quote", users can discover new and inspiring quotes while also enjoying a dynamic and visually stunning background. It's great for personal use or as a tool to provide inspiration to visitors of your website. The ability to share the quote on social media. With a simple click, you will get a fresh quote and a new background, it's an perfect way to start the day.
